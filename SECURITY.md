# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Tree::MultiNode, please report
it responsibly.

**Contact:** Todd Rinaldo <toddr@cpan.org>

You may also file an issue at
<https://github.com/cpan-authors/Tree-MultiNode/issues> if the
vulnerability is not sensitive.

We will acknowledge receipt within 72 hours and aim to provide a fix or
mitigation plan within 30 days.
